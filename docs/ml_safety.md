# ML Safety

The initial model is a placeholder inference pipeline. It is not trained on a validated dermatology dataset and must not be presented as medically accurate.

## Requirements Before Real Deployment

- Train and validate with a legitimate dermatology dataset.
- Evaluate performance across skin tones, lighting conditions, image quality levels, age groups, and device types.
- Measure false negatives carefully because missed high-risk lesions can cause harm.
- Include clinician review and regulatory assessment before any clinical use.
- Maintain audit logs for model version, prediction category, and confidence availability.
